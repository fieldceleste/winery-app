## <center> User Stories

* As a user, I want to see a list/menu of all available kegs. For each keg, I want to see its name, brand, price and alcoholContent. ‚àö
* As a user, I want to submit a form to add a new keg to a list.‚àö
* As a user, I want to be able to click on a keg to see its detail page.‚àö
* As a user, I want to see how many glasses are left in a keg.‚àö
* As a user, I want to be able to click a button next to a keg whenever I sell a glass of it. Glasses cannot go below 0. ‚àö
* As a user, I want the option to edit a keg's properties after entering them just in case I make a mistake. ‚àö
* As a user, I want to be able to delete a keg.‚àö

## <center> Setup/Installation Requirements


#### Install this application -

* Clone this repository: https://github.com/fieldceleste/winery-app.git
* Open the repository folder in the terminal
* Open the folder in editor of choice
* Enter `$npm install `
* Then enter `$npm run build` to build production.
* To run on server type `$npm start`
